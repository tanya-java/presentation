= Java Enterprise

== JavaEE vs Spring Framework

=== JavaEE focus on

[.step]
* Веб-приложения
* Безопасность транзакций
* Администрирование пользователей и данных
* Высокая доступность

=== JavaEE focus on

[.step]
* Безопасность ресурсов
* Коррупционной безопасности
* Масштабируемость
* Четко определенные интерфейсы
* Модульные компоненты
